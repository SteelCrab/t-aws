# EMD

[🇺🇸 English](README.md) | [🇰🇷 한국어](README.ko.md)

`emd`는 AWS 리소스를 탐색하고 포괄적인 마크다운 문서를 생성하기 위해 설계된 터미널 사용자 인터페이스(TUI) 애플리케이션입니다.

## 주요 기능

- **리소스 탐색**: EC2 인스턴스, VPC(네트워크), 보안 그룹, 로드 밸런서를 쉽게 찾아볼 수 있습니다.
- **블루프린터**: 여러 리전과 서비스에 걸친 리소스를 선택하여 하나의 통합된 문서 블루프린트를 만들 수 있습니다.
- **마크다운 생성**: 선택한 리소스에 대한 상세한 마크다운 문서를 자동으로 생성하며, 네트워크 구성도(Mermaid.js)도 포함됩니다.
- **TUI 인터페이스**: `ratatui`로 구축된 사용자 친화적인 터미널 인터페이스를 제공합니다.

## 사용법

애플리케이션 실행:

```bash
cargo build --release
./target/release/emd
```


## 설정

블루프린터 데이터는 다음 경로에 로컬로 저장됩니다:
`~/.emd/blueprints.json`
